Struct DuckDbConnection 

Source 
pub struct DuckDbConnection { /* private fields */ }
Expand description

A Diesel connection to a DuckDB database.

Open with [diesel::Connection::establish], passing either ":memory:" or a file path (with an optional "duckdb://" prefix).
