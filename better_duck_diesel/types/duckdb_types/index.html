Module duckdb_types 

Source
Expand description

DuckDB-specific SQL types not available in Diesel’s standard type set.

Diesel’s built-in SQL types cover the common SQL standard. DuckDB adds several unsigned integer widths, HUGEINT/UHUGEINT, and TIMESTAMPTZ. These types are defined here so that HasSqlType and FromSql/ToSql impls can reference them without colliding with Diesel’s own types.
