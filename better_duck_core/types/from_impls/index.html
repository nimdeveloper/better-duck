Ergonomic From<T> conversions into DuckValue. Ergonomic From<T> conversions into DuckValue.

These impls let you write DuckValue::from(42i32) or "hello".into() instead of constructing the enum variant by hand.

§Note on the read path

from_duckdb_vec is not expressed as a From impl because it requires three inputs (vector, type_id, row_idx) and performs packed-array reads with validity-bitmap checks — a single-input From cannot express it.
