Struct TimeNs 

Source 
pub struct TimeNs(pub NaiveTime);
Expand description

A NaiveTime with nanosecond precision, mapping to DuckDB’s TIME_NS type.

chrono::NaiveTime stores sub-second time as nanoseconds internally, so no precision is lost in the round-trip.
