Chrono date/time types and DuckDialect implementations for the chrono feature. DuckDialect implementations for chrono types using the duckdb_value API.

These conversions operate on duckdb_value heap objects created by duckdb_create_date / duckdb_create_time / etc. They are not used when reading values from data chunk vectors; for that path see DuckValue::from_duckdb_vec (see value.rs).
